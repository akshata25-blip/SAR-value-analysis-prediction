# SAR Value Analysis and Prediction Using SVM and KNN

## Project Overview
This project focuses on analyzing and predicting SAR (Specific Absorption Rate) values using Machine Learning algorithms such as Support Vector Machine (SVM) and K-Nearest Neighbors (KNN).
The system helps in understanding SAR patterns through data preprocessing, analysis, classification, and prediction techniques.

## Features
- Data preprocessing and cleaning
- SAR value analysis
- Prediction using Machine Learning models
- Evalution using MSE and R2 score
- Accuracy comparison of algorithms
- Data visualization and insights

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Machine Learning Algorithms
- Support Vector Machine (SVM)
- K-Nearest Neighbors (KNN)

## Workflow
1. Dataset collection
2. Data preprocessing
3. Feature selection
4. Model training
5. Prediction and evaluation
6. Result visualization

 ## Results
